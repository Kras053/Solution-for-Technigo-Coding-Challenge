Mob programming

Definition:

Mob programming is when a team work together in real time using one computer to write all of the code for the given task.

It is like pair programming but with an entire team, so more like a programming party.

Photo: mtv.com

There are some key positions that can be assigned to the team such as:

the mob: decides what and how to do things
the navigator: gives the mobs instructions to the driver
the driver: puts the instructions into code
the facilitator: makes the mob stay on track and decides when to change roles
